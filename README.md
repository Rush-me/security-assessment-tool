# Security Assessment Tool

A modern, accessible security assessment application built with **Angular 18** and cutting-edge web technologies.

## 🎯 Project Overview

The Security Assessment Tool is a comprehensive Angular 18 application designed to provide security assessments and vulnerability evaluations. Built with modern best practices, it leverages Angular signals for reactive state management and ensures accessibility compliance with WCAG AA standards.

## ✨ Features

- **Modern Angular Architecture**: Built with Angular 18 using standalone components and signals
- **Reactive State Management**: Leverages Angular signals and computed properties for predictable state updates
- **Accessibility First**: Full WCAG AA compliance with AXE accessibility checks
- **OnPush Change Detection**: Optimized performance with OnPush change detection strategy
- **Responsive Design**: Mobile-friendly, responsive UI components
- **Reactive Forms**: Robust form validation using reactive forms patterns
- **Type-Safe**: Strict TypeScript type checking throughout the application

## 🛠️ Technology Stack

- **Angular 18**: Latest Angular framework with signals and standalone components
- **TypeScript**: Strict type checking enabled
- **Reactive Forms**: Advanced form handling and validation
- **CSS**: Modern CSS with flexbox and grid layouts
- **Accessibility**: WCAG AA compliance, AXE testing
